  1. Селектор по элементу: p {} - селектор по тегу p, будет применять свойства ко всем элементам списка p;
  2. Селектор по идентификатору: #title {} - селектор по id "title", будет применять свойства только к одному элементу, который имеет id "title";
  3. Селектор по классу:
    .text {} - селектор по классу "text", будет применять свойства ко всем элементам с классом "text";
    p.text {} - селектор по классу "text" для абзацев "р", будет применять свойства ко всем абзацам (тег "р") с классом "text";
  4. Селектор по нескольким классам: .text.big {} - селектор по классу "text" и "big", будет применять свойства ко всем элементам, которые имеют одновременно класс "text" и "big";
  5. Универсальный селектор: * - применяет свойства ко всем элементам на странице;
  6. Вложенные селекторы: ul li - применяют свойства к элементам, согласно прописанному "пути" вложенности, где ul - родительский элемент, li - дочерний элемент (не все элементы можно вложить один в другой);
  7. Дочерние селекторы: .list > .item - свойства применятся к элементу с классом ".item", который вложен непосредственно в элемент с классом ".list";
  8. Соседние селекторы: i + b - свойства применятся к элементу с тегом b, который по коду идет сразу за элементом с тегом i;

  10. Селекторы атрибутов: [href] - применяются к элементам, у которых есть атрибут [href];
    • a[target="_blank"] - свойства применятся к элементу "a" с атрибутом "target", который равен "_blank";
    • [href^="https"] - свойства применятся к элементам с атрибутом href, название которого начинается с "https";
    • [href$="com"] - свойства применятся к элементам с атрибутом href, название которого заканчивается с "com";
    • [href*="oo"] - свойства применятся к элементам с атрибутом href, название которого содержит "oo";
    • [class~="medium"] - свойства применятся к элементам с атрибутом class, название которого содержит "medium";
    • [class|="medium"] - свойства применятся к элементам с атрибутом class, название которого начинается с "medium", за которым идет дефис, либо именно так и называется;
